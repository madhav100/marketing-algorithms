# Presentation Assets

This folder intentionally keeps presentation content in text/vector formats to avoid binary-file restrictions in code review pipelines.

## Available files
- `customer-behaviour-flow-across-the-commerce-system-outline.md` — full 10-slide narrative content.
- `customer-behaviour-flow-preview.svg` — visual preview board of all 10 slides.

## Why no `.pptx`
Some repository/review workflows reject binary attachments. The deck content is therefore maintained in source-friendly files (`.md` + `.svg`) that can be exported to PowerPoint outside this repository when needed.
