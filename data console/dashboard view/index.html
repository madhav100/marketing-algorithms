Concept Objects

Stream Object: CSV Export

Each file in data cloud/csv-exports/ is treated as a stream source.

Transformer Object: csvDataStream

Reads line-by-line, parses headers/rows, converts primitive values, and emits records.

Storage Object: DataObjectsLake

Creates entity collections by filename and upserts records using id.

Output Object: lake-summary.json

Captures relationships and ingestion status so you can monitor what was processed.
